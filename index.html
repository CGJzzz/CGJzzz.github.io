So called Memory Palace

一种对自己很狠的行为，让大脑疲惫。

记忆宫殿，这个词，在我脑海突然浮现出来，也许这就是对学习的悟性。这个词一早是我从福尔摩斯学过来的，反派用一个特别神奇的能力，记忆力特别强，我到现在能开始理解，从这个物化具体化的记忆宫殿中来理解抽象化的记忆宫殿。电视剧里面福尔摩斯从旋转楼梯往下走，看到了反派的宏伟的记忆宫殿。我到现在终于悟出了何为记忆宫殿以及如何正确利用，并开始尝试。我们往常需要作出较多的努力和更强劲的意志力去克服困难，通过近似无规则的daily routine来复习记忆存储，通过这个记忆宫殿，可以在要求相同的努力跟意志力下更有效有规律去构建我们自己的知识殿堂与体系，which we call it Deleberate Practice。

2015广东卷高考作文题：感知自然(材料作文)

六、本大题1小题，60分。

24.阅读下面的文字，根据要求作文

看天光云影，能测阴晴雨雪，但难逾目力所及；打开电视，可知全球天气，缺少了静观云卷云舒的乐趣。

漫步林间，常看草长莺飞、枝叶枯荣，但未必能细说花鸟之名、树木之性；轻点鼠标，可知生物的纲目属种、迁徙演化，却无法嗅到花果清香、丛林气息。

从不同的途径去感知自然，自然似乎很“近”，又似乎很“远”。

要求：

1，自选角度，确定立意，自拟标题，文体不限

2，不要脱离材料内容及含义的范围

3，不少于800字

4，不得套作，不得抄袭

不得不承认的是，我的语文基础本来是非常差的，因为小学初中高中高二之前遇到的都是男老师，还是类似临时工的那种，这运气，所以非常差。知道自己语文差，就应该换一种方法来弥补。比如说，我记得有一次作文，成绩出来了，老师表扬了一个女同学，她写的是泰戈尔的“生命以痛吻我，我报之以歌”。这种思路，emmmm，说实话，其实我根本就没想过，我脑海想到的就是，小时候看奥特曼一直有个疑问，每次奥特曼被怪兽打的不行了，才放激光杀死怪兽，直到今天，看到一个人的评论才使我豁然开朗，他说：你斗地主直接扔炸弹啊？。我看笑话可是一目十行，记忆深刻。所以我每次写作文都是写段子。事实证明好像还真有五十分，我的阅读拿高分，不存在的。

想象还在课堂演讲，现在总结下，刚才肯定有些童鞋没听清，重点1，关注当年3月份到高考的时事政治；重点2，买高质量的杂志，比如《大学》；重点3，心态要好。遥想高考的那天，我凌晨四点入睡了，早上起来考试，中午，舍友有点吵依旧没睡，第二天，还是三四点没睡，我就是失眠。。。

演技结束，溜了。

总是很喜欢问一些类似1+1=2的问题，其实说到底，是想跟别人说话。

但似乎并不是所有人都愿意跟你说话，所以就是发微信之前，问问自己，不能百度吗。

高中最伤心的一刻

总喜欢想一些有的没的，比如说什么时候可以作为一名优秀的毕业生被请回中学去演讲，去分享自己的学习经验。

想着回去演讲的那一天，我会声情并茂的介绍我们华师SCNU，此时会有个师弟问我：师兄能不能说一说你高中最伤心的一刻呢？

我会想着，该不该告诉他呢？

想了想，班主任突然出去了，最后还是说了，此时泪水先落在镜片上，然后落了下来。说： “是高考的最后一刻，考完了英语，我收拾好了东西，蹦蹦跳跳，注意这个形容词，回到了饭堂。（因为教室都被借去考试了，只能大家在饭堂复习。）我当时是第一个昂首阔步的回到了饭堂的座位上，注意是昂首阔步。我一进饭堂，看到了王老师，就说了句： ”请叫我重本大学生“。寓意是自己会超常发挥。王老师说了句： “欢迎二本大学生”。按照正常逻辑，我是没什么感觉，但我顿时觉得了高三这一年，老师看在眼里，心里清楚什么段位了。

于是班主任在整理等到了所有同学，对所有人说了句“我该叫你们重本大学生。祝你们成绩心想事成。。。“。其实后面他说的我都不太清楚了，原因是我看着他站在桌子上，站在最高的位置跟他们们说着告别的话，但我似乎内心空荡荡的，什么也听不清了，我依稀记得回家的路上，等红绿灯的时候，前面的公交车有个广告写着： 欢迎报考。爸爸妈妈问我话，我也没接。在想什么呢？想到天际了。

总结

今天是29号，这个月其实过的挺糜烂的，反正就是早上就是起不来，都是》10点钟。时间管理一塌糊涂，其实之前也是一塌糊涂。包括今天，还在打游戏到三点钟AM，现在四点钟在写这个月记。

1.时间管理。突然发现番茄时间管理真的挺好用的，如果没有这个东西我怀疑真的看不到一半。现在看完了高数第一章，包括教程，作业，慕课。我现在在想会不会时间不够呢？有点担心，因为我是真的一点没学过，转专业过来之后。

2.能不能玩的问题上？原则上是不能的。但怎么管好自己，防止效率过低呢？

3.背单词进度。

我真的没管理好时间，现在死了都要滚起来背单词。

4.时间管理

最后一周还算是正常的，也体会到了番茄时间管理的好处，希望能完全适应这样的学习时间分配规划。

还有就是先记着把数学全部过一遍，再考虑别的吧。

万物起源

我为什么想读研？

对我自己大一混乱的生活学习状况的总结：

​ 一句话：无奈、绝望。

我先解释一下：为什么挂科？

（我想很多老师很讨厌有很多理由的学生）

第一门是数字逻辑电路。我原专业是经管学院的工商管理，我大一下学期有一门 管理学，因为当时大一下期中我就确认转过来了，觉得原来的专业不用管了，然而我挂了，然后学分认证（就是我在经管修过相应的课可以转为计算机的课程，比如我在经管修了高等数学，学分认证就不需要修计算机的高等数学了，因为我已经学过了），结果他把我管理学认证成数字逻辑电路，我就怪怪的挂了一课。

第二门离散，我没有好好学习，很对不起浪费的时间。

因为挂了科，所以错过了很多的机会。

看到“能够提心个人科研潜质的材料的时候”，我整个人都崩溃了。

我反复想了几天，我挂了科，绩点不好看，我凭什么认为自己能做到研究生的水平？自大吗？

这就像是道证明题，我证明了凭目前的所有情况来说，我不行，我不符合要求。

我想努力，我想读研究生，我想继续学习，因为在经管那一年我学的不开心，那整整一年的夜里睡得跟本不踏实所以失眠。（虽然我转到计算机之后仍然失眠，每晚也是2。3点睡。）

我曾经幻想着我自己在研究生复试里面，到时导师会问我个问题：“你愿意读博吗？”

我会答：“我愿意”，我会哭着回答，我太渴望了。

我想读研，我要读研，我想证明自己，我可以做的更好。

(emmmm,写的不好，自己都感动不了，差评，我还会回来的)

我没有对不起谁，我只想把浪费的时间补回来。

怎么样证明自己是优秀的？

每个人都有为自己辩解的理由，每个老师也听过了很多个理由。

我不想解释了，我用行动来证明，让导师知道我有足够的决心去践行。

在计算机学院读了一年，其实学到了很多，认识了很多志同道合的朋友，但我觉得还没有到头。

弄这个博客反反复复好久了，这篇也改了好多次，写得很烂。

但

就一句：我怎么样证明我是导师想要的学生？不就是坚持到明年你复试我的时候吗？我会继续写这个博客，给你看看，我记录的点点滴滴，让你确信我是OK的。
