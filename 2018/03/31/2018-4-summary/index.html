2018-4-summary

So called Memory Palace

一种对自己很狠的行为，让大脑疲惫。

记忆宫殿，这个词，在我脑海突然浮现出来，也许这就是对学习的悟性。这个词一早是我从福尔摩斯学过来的，反派用一个特别神奇的能力，记忆力特别强，我到现在能开始理解，从这个物化具体化的记忆宫殿中来理解抽象化的记忆宫殿。电视剧里面福尔摩斯从旋转楼梯往下走，看到了反派的宏伟的记忆宫殿。我到现在终于悟出了何为记忆宫殿以及如何正确利用，并开始尝试。我们往常需要作出较多的努力和更强劲的意志力去克服困难，通过近似无规则的daily routine来复习记忆存储，通过这个记忆宫殿，可以在要求相同的努力跟意志力下更有效有规律去构建我们自己的知识殿堂与体系，which we call it Deleberate Practice。

那一天

想着总有一天自己会成为某个学校的教授，某个演讲的人，会想说： 不知道同学们对穷是怎么样的理解呢？我中学掉一支笔在地上就会觉得很可惜。然后在大学大三的时候自己开始独立了，然后掉一支笔在地上的时候想着： 大哥别断啊，亏不起啊。捡起来的时候被自己这个念头笑死。

我想成为让别人骄傲的理由

你们可以认真的看到图中写到字，欢迎华师落榜的学生报考我们香港中文大学（深圳校区），该校区高考分数于2016年已经超过了中大。我很渴望，能够成为很厉害的人。渴望，渴望。

我可以自豪的介绍我们华师应用心理学的强大全国A级学科，与清华北师齐名。我知道这么多年来，一些学校衣依附地势、资金等等投入，得到了我们华师比不赢的资源，但心理学跟师范专业是我校名牌专业，我也想自己更强，成为别人的骄傲。

释然

很感谢这里学到的，我喜欢竞争更激烈的学校，工科蓬勃的学校，喜欢在这样的环境才能更有动力，虽然我这三年也不怎么样。

虽然大学学习在个人，

虽然xxx，

虽然怎么样都好。

在很多比赛也许我们专业并不是很出色，甚至排名落后，被人歧视，比如被网易点名不要实习生，我亲身经历。但我们学校还是拿到了博士点，说明我们学校的成果至少能通过社会的评估，相比广工华农。我认为广工华农的计算机比我们学校的更好。有些学校拿得到博士点有些拿不到，我并不是什么优越性，才发现有时候211还是挺管用的。

修行在个人，自己要更加努力。说的有点乱，这一节。

惊了

换了新手机，但好像静音没听到闹钟，然后舍友在打游戏，问了队员要不要书，我TM潜意识有人问我要不要书，我惊喜的说了句要，然后坐起来了。

捡到个宝

今天捡了一份生科的硕士论文，想摆着鼓励自己也有写硕士学位论文的资格，然后又捡了一本，虽然自己看不懂，但想把这本送给有志科学的师弟吧，希望他也能争取自己的硕士学位。

无题

很难过，在等待把项目部署到服务器的时候，有一些想法浮现在我的大脑。

现在才发现想要一个硕士学位这么难，知道自己很多方面都没有做好，检讨一下。清晰的意思到此时此刻的自己，无论如何都是考不上的，得不到自己想要的，everything。

别人能保研，自己都快考不上，难受。复习。

不安

知道自己考上很渺茫，但想很努力。

自己坐不住，所以想早上4点起来学半个小时，再回去睡，因为自己平时睡不好，更加坐不住，还不如让大脑疲惫，绷紧神经到能坐下来。

1.虽然今天 本月完成了 数学上的复习计划，但是我觉得做不了难题，同时概念不清的现象很严重，我该怎么查漏补缺呢？
